# Python para ciencias sociales

Este repositorio contiene los materiales para el curso "Python para ciencias sociales".

## Estructura del curso

El curso está dividido en 5 sesiones de 3 horas cada una. A continuación se detalla el contenido de cada sesión y los enlaces a los materiales correspondientes.

### Sesión 1: Gramática básica de Python y configuración del entorno
- **Objetivo:** Familiarizarse con el entorno de Jupyter, instalar las herramientas necesarias y aprender los fundamentos de Python.
- **Materiales:**
    - [Notebook para el alumno (Sesión 1)](./sesion1.ipynb)
    - [Notebook para el profesor (Sesión 1)](./sesion1_profesor.ipynb)

### Sesión 2: Flujo de control, funciones avanzadas y manejo de ficheros
- **Objetivo:** Dominar el flujo de control, aprender a escribir funciones más flexibles y manejar ficheros de texto.
- **Materiales:**
    - [Notebook para el alumno (Sesión 2)](./sesion2.ipynb)
    - [Notebook para el profesor (Sesión 2)](./sesion2_profesor.ipynb)

### Sesión 3: Introducción al análisis de datos con Pandas
- **Objetivo:** Aprender a cargar, inspeccionar, limpiar, transformar y filtrar datos del mundo real con la librería Pandas.
- **Materiales:**
    - Notebook para el alumno (Sesión 3) - *Próximamente*
    - [Notebook para el profesor (Sesión 3)](./sesion3_profesor.ipynb)

### Sesión 4: Manipulación de datos y visualización con Seaborn
- **Objetivo:** Aprender a manipular datos con `.groupby()` y a crear visualizaciones con Seaborn.
- **Materiales:**
    - Notebook (Sesión 4) - *Próximamente*

### Sesión 5: Machine learning y proyecto final
- **Objetivo:** Desmitificar el machine learning construyendo un modelo predictivo simple y completar un pequeño proyecto de análisis.
- **Materiales:**
    - Notebook (Sesión 5) - *Próximamente*
